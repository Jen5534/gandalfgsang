Core Experience
Booking that thinks ahead
🎯 Smart Desk Scoring
  • Matches preferred neighbourhood and desk features (standing, dual-monitor, accessible)
  • Boosts scores when teammates are already booked nearby
  • Adjusts for anchor days — anticipates higher occupancy
  • Learns from working pattern (office vs remote days)
📅 Anchor Day Policy
  • Admin sets expected office days per team (e.g. Platform Eng: Wed + Thu)
  • Admin sets site-wide anchor days (e.g. Edinburgh: Wednesday all-hands)
  • Effective days = union of personal + team + site settings
  • Push notification + dashboard alert for any unbooked anchor days in next 14 days
⚡ Walk-in Auto-Booking
  • Building scan-in (NFC / QR) triggers desk allocation if none exists
  • Browser geolocation watches for proximity to office (configurable radius, default 300m)
  • Picks the highest-scoring available desk automatically
  • Confirmed via push notification — one tap to accept or dismiss
🔄 Friction-Free Change
  • Shorten a full-day booking to AM or PM with one tap
  • Swap to any other available desk, same day or future
  • Hand-over to a teammate — checks their existing bookings for conflicts first
  • Freed capacity is immediately back in the pool for others to book

Azure Policy & Defender
Enterprise Security
Zero Trust. Defence in depth. By design.
Authentication
Azure AD / Entra ID with SAML 2.0 / OIDC. MFA enforced for all users. Conditional Access blocks non-compliant devices.
Authorisation (RBAC)
Three roles: User, Manager, Admin. Fine-grained API permissions via OAuth 2.0 scopes. Principle of least privilege throughout.
Encryption
TLS 1.3 in transit (HSTS enforced). AES-256 at rest for all Cosmos DB data. Key Vault manages all secrets — no plaintext in code or config.
Audit Logging
All booking events, admin actions, and auth events streamed to Azure Monitor. Sentinel integration for SIEM alerting on anomalies.
Network Security
Azure Front Door WAF with OWASP 3.2 rule set. DDoS Protection Standard. Private endpoints for all backend services — no public database exposure.
API Security
Azure API Management enforces OAuth 2.0 bearer tokens, rate limiting (per user and per IP), and request validation on all endpoints.
Penetration Testing
OWASP Top 10 coverage in CI/CD via SAST (SonarQube) and DAST (OWASP ZAP). Annual external pen test. Dependency scanning on every PR.
Compliance & Privacy
GDPR by design: data minimisation, right-to-erasure API, retention policies (booking data purged after 13 months). UK data residency. ISO 27001 aligned controls.
