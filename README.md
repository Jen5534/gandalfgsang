# Perch — Smart Desk Booking

**Team:** Gandalf's Gang &nbsp;|&nbsp; **Code:** T-005 &nbsp;|&nbsp; **Captain:** Paul Bednall

## What we built

Perch is an AI-powered desk booking and workplace management tool built for large hybrid organisations. Employees declare their office days, and a smart allocation engine automatically assigns the best available desk the night before — factoring in team proximity, personal preferences, historic patterns, and manager-configured seating rules. It replaces the daily scramble for desks with a system that works quietly in the background, so people arrive knowing exactly where they're sitting.

## Running the demo

```bash
node server.js
```

Then open **http://localhost:3000** (app) or **http://localhost:3000/admin.html** (admin portal, PIN: `1234`).
